This program works similar to Jarvis from the Marvel movies with Iron Man. You first need to start by saying Jarvis then ask your
question and jarvis will answer. Currently he only works with asking questions but there are many more fetures to come!!

Requirements:
    installs:
        openai
        pyttsx3
        speech_recognition
    In the get input file make sure to put in your API key in the commented section
